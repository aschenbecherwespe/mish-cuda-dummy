# mish-cuda-dummy

Drop-in replacement/dummy package for mish-cuda (`mish_cuda`) in case install/compilation of [`mish-cuda`](https://github.com/thomasbrandon/mish-cuda) fails/takes too long.

**NB!:** This is not performance optimized. Use it for e.g. neural network architecture/model exploration only!


**Setup**

Clone this repo and run `pip install -e .`
